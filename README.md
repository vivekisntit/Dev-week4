## Sales Data Analysis – Project Documentation

### Project Overview

This project performs sales data analysis using python, pandas and matplotlib.The analysis examines sales performance across regions, products and monthly trends. The workflow includes loading the dataset, cleaning the data then performing proper analysis and visualizing insights using charts.

---
### Technical Requirements Implemented
- Used pandas to load and analyze the dataset from a CSV file.
- Checked for missing values and verified dataset integrity.
-Calculated multiple sales metrics including averages, medians, and total revenue.
-Generated insights such as region-wise and product-wise sales.
- Created visualizations using matplotlib.
Added comments in the code to explain each step of the analysis.
---
### Project Structure
The project has the following structure:

#### Data Loading
Sales data was loaded from a CSV file using pandas.
#### Data Cleaning & Validation
- Dataset structure and data types were verified.
- Date values were converted into datetime format.
- Data validation ensured correct numeric values for quantity and sales.
#### Data Analysis
Several key metrics were calculated such as:
- Total overall revenue 
- Average overall revenue 
- Quantity region-wise
- Product-wise sales and quantity
- Monthly sale trends
#### Data Visualization
The analysis includes multiple chart types created using matplotlib:
- Pie Charts for distribution of sales across regions, products and months.
- Bar Charts for comparing quantities and average sales across regions.
-Line Charts for identifying trends in sales across regions and time.
